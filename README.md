# 🚦 Real-Time Traffic Congestion Prediction

AI-powered traffic risk prediction system using:
- Machine Learning
- Deep Learning (Future Scope)
- TomTom Traffic API
- Streamlit Dashboard

## Features
- Single City Traffic Dashboard
- Multi-City Risk Comparison
- India Heatmap Visualization
- PDF Report Generation
- Risk Ranking System

## Tech Stack
- Python
- Streamlit
- Scikit-Learn
- Folium
- Plotly
- TomTom API

## Deployment
Deployed using Streamlit Cloud
